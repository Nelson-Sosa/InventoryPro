# Documentación de Uso de IA (Resumen)

## Herramienta Utilizada
- **ChatGPT (GPT-4 / GPT-4o)**  
Usada para asistencia en desarrollo frontend, generación de ejemplos, solución de errores y optimización de código.

## Cómo se Usó la IA
- Creación de componentes React.js con TypeScript y Tailwind CSS
- Sugerencias de arquitectura de carpetas y rutas
- Generación de ejemplos de consumo de APIs simuladas con `json-server`
- Manejo de estado y validaciones de formularios
- Corrección de tipados y adaptación a buenas prácticas

## Proceso Breve
1. Se planteó un prompt inicial con el objetivo deseado.
2. Se probó el código generado y se corrigieron errores menores.
3. Se refinó el prompt hasta obtener un resultado funcional y limpio.
4. Se implementó directamente en el proyecto.

## Mis Habilidades Aplicadas
- Frontend: React.js, TypeScript, Tailwind CSS
- Consumo de APIs simuladas con `json-server`
- Manejo de estado y validaciones de formularios
- Uso de IA para acelerar desarrollo y mejorar buenas prácticas
